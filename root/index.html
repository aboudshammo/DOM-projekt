Introduktion

Dokumentobjektsmodellen (DOM) är data representationen för objekten av strukturen och innehållet i ett dokument på webben. I den här guiden presenterar vi kort om hur DOM fungerar. Vi tittar på hur DOM representerar ett HTML dokument i minnet och hur Api fungerar för att skapa webbinnehåll och applikationer.

DOM är ett gränssnitt för HTML och säger till att program kan ändra strukturen, stil och innehåll i ett HTML /CSS dokument. Dom består av noder och objekt och på så sätt kan programmeringsspråk som t.ex javascript modifiera html/CSS dokumentet. DOM kan repsenteras visuellt på det sättet :

Det som DOM gör att att den delar upp strukturen i form av objekt, alla element, text, attribute och annat innehåll delas upp och kan sedan med hjälp av javascript modifieras. Javascript kan med hjälp av DOM ändra :

  • alla HTML-element på sidan
  • alla HTML-attribute på sidan
  • alla CSS-stilar på sidan
  • ta bort befintliga HTML-element och attribut
  • lägga till nya HTML-element och attribut
  • reagera på alla befintliga HTML-händelser på sidan
  • skapa nya HTML-händelser på sidan

Hur använder man DOM ?

För att använda sig av DOM man måste känna till APIs alltså funktioner som man kan använda för att programmera eller få tillgång till html eller css dokumentet, dessa funktioner kan variera med här är några exempel :

  • element.setAttribute()
  • element.getAttribute()
  • document.setElement()
  • document.getElement()

Dessa funktioner kan användas för att returnera eller ändra element och attribut, det finns även massa andra funktioner som kan ge mer flexibilitet och mer ändringar i html/css dokumentet.

I Vilka sammanhang kan DOM och javascript användas ?

 

<!DOCTYPE html>
<html>
<body>

<h2>My First Page</h2>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "Hello World!";
</script>

</body>
</html>

Detta är ett sammanhang där Javascript används med hjälp av DOM för att ersätta “Demo” med “Hello World” med hjälp av funktionen document.getElement.Det finns massor med sammanhang där man kan använda sig av dom och javascript, i den här sidan har vi kodat knappar som kan byta bakgrund och färg på text.
